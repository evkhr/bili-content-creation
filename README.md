# B站内容发布助手 (bili-content-creation)

<p align="center"><img src="icon.png" width="150" alt="icon"></p>

基于 B站开放平台 API 的自动化内容投稿工具，支持视频上传、封面管理、视频去重、批量发布等功能。

## 功能

- **视频投稿** — 通过开放平台 API 实现视频上传与发布
- **封面管理** — 自定义封面上传
- **视频去重** — 基于 SHA-256 内容指纹的快速/深度去重
- **批量发布** — 多账号并发投稿
- **内容管理** — 后续支持专栏图文等功能

## 技术栈

- Node.js / Electron
- B站开放平台 API (open.bilibili.com)
