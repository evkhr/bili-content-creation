# B站内容发布助手 (bili-content-creation)

<p align="center"><img src="icon.png" width="150" alt="icon"></p>

基于 B站开放平台 API 的内容创作辅助工具，支持视频上传、封面管理、内容去重等功能。

## 功能

- **视频上传** — 通过开放平台 API 上传视频稿件
- **封面管理** — 上传与设置视频封面
- **内容去重** — 基于文件指纹的重复内容检测
- **内容管理** — 支持视频及后续专栏图文等功能

## 技术栈

- Node.js / Electron
- B站开放平台 API (open.bilibili.com)
